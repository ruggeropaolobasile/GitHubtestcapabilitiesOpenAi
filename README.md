# GitHubtestcapabilitiesOpenAi

Repository di test per dimostrare in modo pratico le capacità del connettore GitHub con ChatGPT/OpenAI.

## Obiettivi

- verificare lettura file
- testare ricerca nel codice
- simulare analisi repository
- provare workflow branch/commit/PR
- usare issue e commenti come superficie di test

## Struttura iniziale

- `README.md` : overview del repository
- `.gitignore` : esclusioni comuni
- `docs/capabilities.md` : casi d'uso da provare con il connettore

## Prompt di esempio

- Mappami il repository
- Spiegami lo scopo dei file presenti
- Cerca tutti i riferimenti a capability o connector
- Proponi una struttura più completa
- Crea una branch per aggiungere esempi

## Stato

Seed iniziale creato per bootstrap del repository.
