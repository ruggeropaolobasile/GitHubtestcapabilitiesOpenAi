# Demo prompts for GitHub connector

Use these prompts against this repository to validate connector behavior.

## Discovery
- Mappami il repository.
- Elenca i file più importanti.
- Spiegami come è organizzata la cartella `.github`.

## Code reading
- Analizza `examples/node/sample-api.js`.
- Spiegami cosa fa `examples/python/sample_service.py`.
- Cerca tutti i riferimenti a `capabilities`.

## Review and change management
- Riassumi la differenza tra `main` e `feat/connector-demo`.
- Dimmi se questa branch è pronta per PR.
- Analizza la PR come farebbe un senior reviewer.

## Evolution
- Proponi test mancanti.
- Proponi una cartella `tests/`.
- Suggerisci una roadmap per rendere il repo una vera demo pubblica.
